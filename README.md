# spiceUp is my first project where I used the CSS framework TailwindCSS and decided to integrate it with React for the UI. This was a bit challenging at first to get everthing set up and working properly, setting up custom Tailwind properties in the .config file and getting it to work well with react. After getting used to the learning curve, the developer experience was very pleasant and I can safely say that I will be looking forward to using TailwindCSS with React in the future. The functionality of the app goes as follows:
## You are Greeted with the main Page that has a featured ingredient displaying dishes in a 4x4 grid layout.
![spice_thumbnail](https://user-images.githubusercontent.com/88985587/152846255-6017fba7-86e0-473b-8cc0-2e9e1480af05.jpg)
## Each grid item has a button that adds the dish to your favorites, once clicked you have the option to remove that dish from the favorties state. You can view the dishes you favorited by navigating via the sidebar menu and clicking the favorites button.
